This extension lets you hide distractions on YouTube, Facebook, Twitter, Instagram, Linkedin, and Google.com. It's open source and collects zero data - you can find the source code on GitHub.

To turn it on, open Safari and go to one of the target sites. Then:
1) In the toolbar tap 'Aa' -> 'Manage Extensions', and toggle the extension on.
2) Tap the extension and allow access
